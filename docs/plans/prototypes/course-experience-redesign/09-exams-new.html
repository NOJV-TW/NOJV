5

Proctoring 防作弊

適合關鍵性考試。非強制，可只啟用需要的。

分頁鎖
學生按下「Start exam」後，整個分頁鎖定在考試頁面。離開考試路徑（切其他頁、點 side nav）會自動 307 redirect 回考試。交卷、時間到、或老師手動釋放才解鎖。
無法阻止學生開新分頁 / 新視窗——那是人工監考的責任。
IP 綁定（首次登入）
第一次看到的 IP 被記錄為這位學生的「考試 IP」，之後用不同 IP 提交會被擋下。適合教室內監考場景。
違規處理方式
IP 白名單
只允許指定網段的連線進行考試。離開校園網路就進不來。空白名單將完全禁止進入——這是刻意的 fail-closed 設計。
